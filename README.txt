STEPS TO RUN:
    1. RUN "npm install" FROM THE GAMEOFLIFE DIRECTORY TO INSTALL DEPENDENCIES.
    2. RUN "npm run dev " + THE CONFIGURATION YOU WANT TO USE.
    3. CONFIGURATIONS ARE IN AN ARRAY CALLED "startingCells" WITH COORDINATES IN IT, LOCATES IN THE "initialInput.json" FILE. "0" IS THE FIRST CONFIGURATION.
    4. ACCESS "localhost:3000/" IN YOUR BROWSER

NOTES:
    - "input.json" IS WHERE THE FINAL RESULTS ARE STORED IN.
    - TO CHANGE THE GRID SIZE, NUMBER OF GENERATIONS TO RUN OR ADD CONFIGURATIONS, ACCESS "initialInput.json".
